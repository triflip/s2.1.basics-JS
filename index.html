1. Arrow Functions

Exercise 1: Function conversion

You have a function add(a, b) { return a + b; } that takes two parameters and returns their sum. Convert it into an arrow function. 


const add = (a, b) => a + b;

console.log(add(2, 9));

Exercise 2: Arrow function without parameters

Create an arrow function called randomNumber that takes no parameters and returns a random number between 0 and 100. 


const randomNumber = () => Math.floor(Math.random() * 100);

console.log(randomNumber());

Exercise 3: Using 'this' in arrow functions

Create a class Person with a name property and a greet method using an arrow function. The method should return a greeting that includes the person's name. 


class Person {
  constructor(name) {
    this.name = name;
    this.greet = () => `Hi! ${this.name}`;
  }
}

const user = new Person("Miquel");

console.log(user.greet());

Exercise 4: Arrow function inside a loop

Create a function called printNumbers that takes an array of numbers and uses a for loop to print each number using an arrow function. 


const numbers = [1, 2, 3, 4];

const printNumbers = function(arr) {
  for (let index = 0; index < arr.length; index++) {
    const showNumber = (num) => console.log(num);
    showNumber(arr[index]);
  }
};

printNumbers(numbers);

Exercise 5: Arrow function with setTimeout

Create an arrow function that prints a message to the console after waiting 3 seconds. 


const showMessage = () => {
  setTimeout(() => {
    console.log("See u soon");
  }, 3000);
};

console.log("Bye!");

showMessage();

2. Ternary Operator

Exercise 1: Basic ternary usage

Create a function canDrive that takes an age as a parameter and uses the ternary operator to determine if the user can drive. If the age is 18 or older, return "Can drive"; otherwise, return "Can't drive". 


const canDrive = (age) => {
  console.log(age >= 18 ? "Can drive" : "Can't drive");
};

canDrive(44);

Exercise 2: Comparison with ternary

Write an expression using the ternary operator to determine which of two numbers (num1 and num2) is greater. If they are equal, return "Both numbers are equal". 


const num1 = 19;
const num2 = 19;

const smallBigNumber = 
  num1 === num2 ? "Both numbers are equal" :
  num1 > num2 ? "Num1 is bigger" : "Num2 is bigger";

console.log(smallBigNumber);

Exercise 3: Chained ternary and max value

Use chained ternary operators to determine if a number is positive, negative, or zero. Then, create a function findMaxValue that takes three parameters (a, b, c) and uses ternary logic to find the maximum value. 


let num = -5;

const negativePositive = 
  num === 0 ? "The number is zero" :
  num > 0 ? "The number is positive" : "The number is negative";

console.log(negativePositive);

function findMaxValue(a, b, c) {
  console.log(
    a > b && a > c ? "A is the maximum value" :
    b > a && b > c ? "B is the maximum value" : "C is the maximum value"
  );
}

findMaxValue(4, 8, 32);

Exercise 4: Ternary inside a loop

Create a function oddEven that takes an array of numbers and uses a loop to iterate through it. Inside the loop, use the ternary operator to determine whether each number is even or odd. 


const arrayNumbers = [2, 67, 94, 38, 79, 53, 22];

function oddEven() {
  for (let i = 0; i < arrayNumbers.length; i++) {
    let result = arrayNumbers[i] % 2 === 0
      ? `The number ${arrayNumbers[i]} is even`
      : `The number ${arrayNumbers[i]} is odd`;
    console.log(result);
  }
}

oddEven();

3. Callbacks

Exercise 1: Basic callback

Create a function called process that takes a number and a callback function as parameters. The function should invoke the callback, passing the number as an argument. 


function process(number, cb) {
  cb(number);
}

function callBack(n) {
  console.log(`The number is ${n}`);
}

process(22, callBack);

Exercise 2: Callback with math operations

Create a function called calculator that takes two numbers and a callback function. The callback should be invoked with both numbers. Then, call calculator with a function that adds the two numbers. 


function calculator(n1, n2, cb) {
  cb(n1, n2);
}

function numbers(num1, num2) {
  console.log(`The sum of ${num1} and ${num2} is ${num1 + num2}`);
}

calculator(5, 9, numbers);

Exercise 3: Callback in asynchronous function

Create a function called waitAndGreet that takes a name and a callback. The function should wait 2 seconds and then invoke the callback with the name. 


function waitAndGreet(name, cb) {
  console.log("Waiting for two seconds...");
  setTimeout(() => {
    cb(name);
  }, 2000);
}

function person(name) {
  console.log(`Hi ${name}!`);
}

waitAndGreet("Erik", person);

Exercise 4: Callback with arrays

Create a function called processElements that takes an array and a callback. The function should invoke the callback for each element in the array. 


const elements = [1, 2, 3, 4];

function processElements(arr, cbk) {
  cbk(arr);
}

function loop(array) {
  for (let index = 0; index < array.length; index++) {
    console.log(array[index]);
  }
}

processElements(elements, loop);

Exercise 5: Callback with string transformation

Create a function called processString that takes a string and a callback. The function should convert the string to uppercase and then invoke the callback with the transformed string. 


const charString = "ljtmp";

function processString(stg, cb) {
  let toUpper = stg.toUpperCase();
  cb(toUpper);
}

function toUpperC(result) {
  console.log(result);
}

processString(charString, toUpperC);

4. Rest & Spread Operators

Exercise 1: Spread in arrays

Create two arrays, array1 and array2. Use the spread operator to create a third array that includes all elements from both. 


const array1 = ["car", "plane", "cycle", "ship"];
const array2 = [1, 2, 3, 4];

const arrayMix = [];
arrayMix.push(...array1, ...array2);

console.log(arrayMix);

Exercise 2: Rest in functions

Create a function called suma that uses the rest operator to accept an indefinite number of arguments and returns their sum. 


function suma(...nums) {
  const total = nums.reduce((acc, val) => acc + val, 0);
  return total;
}

console.log(suma(4, 6, 5, 9));

Exercise 3: Copying objects with spread

Create an object object1. Then create object2 as a copy using the spread operator. Modify a property in object2 and confirm that object1 remains unchanged. 


const object1 = { name: "Lena", age: "20", city: "Barcelona" };
const object2 = { ...object1 };

object2.age = 21;

console.log("Original:", object1);
console.log("Modified copy:", object2);

Exercise 4: Rest in destructuring

Create an array with several elements. Use destructuring and the rest operator to assign the first two elements to variables and the remaining elements to a third variable. 


const colors = ["red", "blue", "green", "yellow", "purple"];

const [first, second, ...rest] = colors;

console.log("First color:", first);
console.log("Second color:", second);
console.log("Rest:", rest);

Exercise 5: Spread in function calls

Create a function that accepts three arguments. Then create an array with three elements and call the function using the spread operator. 


function showFruits(a, b, c) {
  console.log(a, b, c);
}

const fruits = ["orange", "watermelon", "apple"];

showFruits(...fruits);

console.log("Showing with:", fruits);

Exercise 6: Merging objects with spread

Create two objects with different properties. Use the spread operator to merge them into a new object. 


const personalInfo = { name: "Jan", city: "Barcelona" };
const professionalInfo = { job: "Football player", position: "Pivot" };

const fullProfile = { ...personalInfo, ...professionalInfo };

console.log("Combined profile:", fullProfile);

5. Array Transformations

Exercise 1: Map

Given an array of numbers [1, 2, 3, 4], create a new array containing the square of each number. 


const nums1 = [1, 2, 3, 4];
const quadrats = nums1.map(n => n * n);

console.log(quadrats);

Exercise 2: Filter

Given an array of numbers [1, 2, 3, 4], create a new array that only contains the even numbers. 


const nums2 = [1, 2, 3, 4];
const parells = nums2.filter(n => n % 2 === 0);

console.log(parells);

Exercise 3: Find

Given an array of numbers [1, 10, 8, 11], use find to get the first number greater than 10. 


const nums3 = [1, 10, 8, 11];
const superiorDeu = nums3.find(n => n > 10);

console.log(superiorDeu);

Exercise 4: Reduce

Given an array of numbers [13, 7, 8, 21], use reduce to calculate the total sum. 


const nums4 = [13, 7, 8, 21];
const suma = nums4.reduce((acc, val) => acc + val, 0);

console.log(suma);

Exercise 5: Chained transformations

Given an array [1, 3, 7, 10, 15, 17, 11, 5, 8, 12, 9], create a one-liner function that filters numbers ≥ 10, multiplies them by 2, and returns the sum of the results. 


const nums5 = [1, 3, 7, 10, 15, 17, 11, 5, 8, 12, 9];
const resultat = nums5
  .filter(n => n >= 10)
  .map(n => n * 2)
  .reduce((acc, val) => acc + val, 0);

console.log(resultat);

Exercise 6: Every / Some

Use every and some to check if all or some elements in the array [11, 12, 13, 14] are greater than 10. 


const nums6 = [11, 12, 13, 14];

console.log(nums6.every(n => n > 10));
console.log(nums6.some(n => n > 13));

Use forEach to print each name in the array ['Anna', 'Bernat', 'Clara'] to the console. 


let noms = ['Anna', 'Bernat', 'Clara'];

noms.forEach(nom => console.log(nom));

Exercise 2: for-of loop

Use a for-of loop to print each name in the array ['Anna', 'Bernat', 'Clara']. 


let noms = ['Anna', 'Bernat', 'Clara'];

for (let nom of noms) {
  console.log(nom);
}

Exercise 3: Filter even numbers

Use filter to create a new array containing only even numbers from [1, 2, 3, 4, 5, 6]. 


let numeros = [1, 2, 3, 4, 5, 6];
let parells = numeros.filter(n => n % 2 === 0);

console.log(parells);

Exercise 4: for-in loop

Use a for-in loop to print each key and its value from the object { nom: "Ona", edat: 25, ciutat: "Barcelona" }. 


let obj = { nom: "Ona", edat: 25, ciutat: "Barcelona" };

for (let clau in obj) {
  console.log(clau, obj[clau]);
}

Exercise 5: for-of with break

Use a for-of loop to print numbers from the array [1, 2, 3, 4, 5, 6] until the number 5 is found, then stop the loop. 


let nums = [1, 2, 3, 4, 5, 6];

for (let num of nums) {
  if (num === 5) break;
  console.log(num);
}

Exercise 6: for-of with index

Use a for-of loop to print each element of the array ['Anna', 'Bernat', 'Clara'] along with its index. 


let names = ['Anna', 'Bernat', 'Clara'];
let index = 0;

for (let nom of names) {
  console.log(`Position ${index}: ${nom}`);
  index++;
}

7. Promises & Async/Await

Exercise 1: Creating a Promise

Create a promise that resolves after 2 seconds and returns the string 'Hola, món'. 


const promesa = new Promise((resolve) => {
  setTimeout(() => {
    resolve("Hola, món");
  }, 2000);
});

Exercise 2: Using a Promise

Use the promise created in Exercise 1. Add a .then to print the result to the console. 


promesa.then(resultat => console.log(resultat));

Exercise 3: Promise with reject

Create a promise that resolves after 2 seconds if the input is 'Hola', and rejects otherwise. 


function saluda(input) {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      input === "Hola"
        ? resolve("Salutació acceptada")
        : reject("Entrada incorrecta");
    }, 2000);
  });
}

Exercise 4: Using async/await

Create an asynchronous function that uses await to wait for the result of the promise created in Exercise 1, and then prints it to the console. 


async function mostraSalutacio() {
  const resultat = await promesa;
  console.log(resultat);
}

mostraSalutacio();

Exercise 5: Error handling with async/await

Modify the function from Exercise 4 to catch any possible errors using a try/catch block. 


async function gestionaSalutacio() {
  try {
    const resultat = await saluda("Hola");
    console.log(resultat);
  } catch (error) {
    console.error("Error:", error);
  }
}

gestionaSalutacio();

Exercise 6: Promise.all

Create two promises that resolve after 2 and 3 seconds, respectively. Use Promise.all to wait for both to resolve and print the results to the console. 


const p1 = new Promise(resolve => setTimeout(() => resolve("Primer"), 2000));
const p2 = new Promise(resolve => setTimeout(() => resolve("Segon"), 3000));

Promise.all([p1, p2]).then(resultats => console.log(resultats));
